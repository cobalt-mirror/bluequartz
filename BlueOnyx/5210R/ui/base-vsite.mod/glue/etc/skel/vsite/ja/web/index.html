これはホームページのサンプルです。 ご自分のホームページで置き換えてください。

ホームページを作るには．．．

お気に入りの FTP ソフトウェアを使って、ホームページ用のファイルを Sun Cobalt サーバにアップロードしてください。
 
ヒント ホームページ用のファイルには、index.html という名前を付けて下さい。

サイトアクセス

サイト管理者と一般ユーザは、次の URL からサイトへログインできます。

　　 http://[DOMAIN]/login/

この URL を開くと、サイトへのログイン画面が表示されます。サイト管理者は、サイト管理者用のユーザ名とパスワードでログインしてください。一般ユーザは、ログインすると自分のアカウント情報を管理できます。
 
ヒント この URL をブックマークしておくと、簡単にアクセスできて便利です。
ユーザのウェブサイト
このサイトに登録されている一般ユーザは、次の URL を使って自分のホームページを作ることができます。

　　http://[DOMAIN]/~<username>/
 
ヒント サイト管理者は、各登録ユーザに各自のアカウント情報の管理・変更方法とホームページの作り方についてご案内ください。
