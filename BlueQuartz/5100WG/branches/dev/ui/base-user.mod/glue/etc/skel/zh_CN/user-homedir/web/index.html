欢迎访问我的主页
这是一个最终用户主页模板。该模板 可引导您创建自己的主页。此页面中可包括与其他人进行联系时所必需的任何类型的信息。其中的主题 可包括有关这些方面的信息：您目前正在处理的项目、项目计划、兴趣、嗜好，以及相关用户、组 或希望与其他人共享的信息链接。您可使用此服务器附带的任何工具或 任何标准的 HTML 编辑器来修改该页面。记住一定要删除 诸如此段落的模板信息。
目前的项目―
插入您目前正在处理的项目列表。如果希望详细解释 每个项目，还可创建其他单独的页面。您可在该部分创建至其他页面的链接。
我的计划―
插入您目前的计划。例如，列出您可能休息的天数或小时数，或白天能找 到您的最佳时间。如果希望详细解释 每一项，还可创建其他单独的页面。您可在该部分创建至其他页面的链接。


喜爱的 WEB 站点―
插入希望与其他人共享的喜爱的 web 站点链接。您可链接 至任何 URL、服务器上已有的任何文件，或将文件上载到服务器，然后再链接至该文件。
